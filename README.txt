# Alien-Hunter-1.0

Welcome to Alien Hunter 1.0 
it's a simple 2D game that I made for my university project


Note 
You must have mysql server running and make a database named "alien hunter 1.0" to use the database of the game
you'll also have to do the following 
make a table named playersinfo into the following database
make four columns named " username,password,maxscore,killcount,maxlevel " where username being the primary key
